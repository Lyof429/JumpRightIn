# Jump Right In! #

## About ##

Instead of going to the main menu, Jump Right In will send you to the specified world in the config file! Launch the game once to generate this file, then specify the world name in the config and presto!

You can either have this world set to load you in on only the first time you launch the modpack, or every time you launch the modpack.

A great utility for packers who want to teach players about their modpack via tutorial worlds and whatnot, or for those who just want to get right into the action!

Developed by MrBysco, requested by Nekomaster1000!
